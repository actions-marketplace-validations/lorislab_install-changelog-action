## Example usage

```
- uses: lorislab/install-changelog-action@v1
- run: changelog version
```

```
- uses: lorislab/install-changelog-action@v1
  with:
    version: "1.6.0"
- run: changelog version
```

## Development

#### Local development

Install the dependencies

```bash
npm install
```

#### Package for distribution

Run prepare

```bash
npm run prepare
```
